# 🚦 TRAFFICAI – AI-Powered Traffic Signal Optimization System

## 🚀 Overview
TRAFFICAI is an AI-based smart traffic management system that dynamically adjusts traffic signal timings based on real-time vehicle density. The system uses computer vision and machine learning to reduce congestion, waiting time, and fuel consumption.

---

## 🎯 Problem Statement
Traditional traffic signals operate on fixed timers, leading to:
- Traffic congestion  
- Fuel wastage  
- Increased waiting times  
- Poor emergency response  

---

## 💡 Solution
TRAFFICAI uses AI and real-time video analysis to:
- Detect and count vehicles per lane  
- Analyze traffic density  
- Dynamically optimize signal timings  
- Provide live monitoring through a dashboard  

---

## 🧠 Key Features
- 🚗 Real-time vehicle detection using YOLOv8  
- 📊 Traffic density analysis  
- 🚦 Dynamic signal control  
- 📡 IoT-based architecture  
- 📈 Live monitoring dashboard  
- 🚑 Emergency vehicle prioritization (future scope)  

---

## ⚙️ Technologies Used
- **Python** – Core programming  
- **YOLOv8** – Vehicle detection  
- **OpenCV** – Video processing  
- **AWS IoT** – Cloud integration  
- **React.js** – Dashboard UI  
- **Edge Devices** (Raspberry Pi / Jetson Nano)
